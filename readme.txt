== Description ==
You need to add a widget to help other developers. Here is a nice way to help some web developers.

Editing .htaccess is sometimes a headache to others. Here My .htpasswd Generator Widget helps you to add a .htpasswd protected folder with easy
You can easly configure this plugin to your site.

== Installation ==
From your WordPress dashboard
1. Visit ‘Plugins > Add New’
2. Search for ‘My.htpasswd Generator Widget’
3. Activate My.htpasswd Generator Widget from your Plugins page.
4. Use shortcode [htpasswd] inside post or page
5. Or you can use htpasswd widget in any widget area
From WordPress.org
1. Download ‘My.htpasswd Generator Widget’
2. Upload the ‘my-htpasswd-generator-widget’ directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)
3. Activate My.htpasswd Generator Widget from your Plugins page.
4. Use shortcode [htpasswd] inside post or page
5. Or you can use htpasswd widget in any widget area

== Frequently Asked Questions ==
1. What the purpose of this plugin?
This plugin provides a simple form to create .htpasswd for protecting any file or folder in apache server.
2. What are the ways to show the .htpasswd creating form?
The plugin supports two methods, either using shortcode or widget. We can place [htpasswd] on any page or post or you can choose the widget .htaccess Modifier
